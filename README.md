# TodoApp

## Getting Started

### Clone this repo

```bash
$ git clone https://github.com/GoodStar20/MERN-Typescript-TodoApp.git
```

### Install dependencies

#### Back-end

```bash
$ cd MERN-Typescript-TodoApp/back-end
$ npm install

$ npm start
```

#### Front-end

```bash
$ cd MERN-Typescript-TodoApp/back-end
$ yarn

$ yarn start
```
